BurnedPlayer - Embeddable Video Player

Method 1: Auto-initialization with data attributes

Simply add the script and use data attributes:

<!-- Include BurnedPlayer script -->
<script src="burned-player.js"></script>

<!-- Create player with data attributes -->
<div data-burned-player 
     data-src="https://your-stream-url.m3u8"
     data-autoplay="false"
     style="width: 800px; max-width: 100%;">
</div>

Method 2: JavaScript initialization

Initialize programmatically for more control:

<!-- Include BurnedPlayer script -->
<script src="burned-player.js"></script>

<!-- Create container element -->
<div id="my-player"></div>

<script>
// Initialize player
const player = new BurnedPlayer('#my-player', {
    src: 'https://your-stream-url.m3u8',
    autoplay: false,
    controls: true,
    width: '100%',
    height: 'auto'
});

// Use player API
player.play();
player.pause();
player.setVolume(0.5);
player.seek(30);
</script>

Options

Available configuration options:

const player = new BurnedPlayer(container, {
    src: 'https://your-stream.m3u8',    // HLS stream URL
    autoplay: false,                    // Auto-start playback
    controls: true,                     // Show player controls
    width: '100%',                      // Player width
    height: 'auto'                      // Player height
});

API Methods

Control the player programmatically:

player.play()              // Start playback
player.pause()             // Pause playback  
player.setSource(url)      // Change stream URL
player.setVolume(0.5)      // Set volume (0-1)
player.seek(30)            // Seek to 30 seconds
player.destroy()           // Clean up player
