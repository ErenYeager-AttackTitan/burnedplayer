VideoPlayer.js - Standalone Usage

How to integrate into any website:

1. Include the script

<script src="https://YOUR_REPLIT_URL/dist/videoplayer.min.js"></script>

2. Add HTML container

<div id="my-video-player"></div>

3. Initialize with JavaScript

const player = new VideoPlayer('#my-video-player', {
  autoPlay: false,
  controls: true,
  volume: 0.8
});

player.loadSource([
  { src: 'video.mp4', type: 'video/mp4' }
]);

4. Or use data attributes (auto-init)

<div data-video-player='{"autoPlay": false, "controls": true}'>
  <video>
    <source src="video.mp4" type="video/mp4">
  </video>
</div>
